About me

Tic Tac Toe

Tic Tac Toe was a project based more solidly on front end design with the exploration of jQuery as the key component of the exercise. It is built completely in javascript with plans to one day incorporate a back end to log scores.

You can find the github repository for it here: Tic Tac Toe

Music Room

Music room was my first introduction to Ruby. It is an app which works to bring muscians within the local area together. It works using a gem named geocoder which logs their ip address on sign in and matches anyone else within the area. The app also has a built in chat so users can begin chatting immediately

You can find the github repository for it here: Music Room

YoungHort Aus

YoungHort Aus is a site built for a small not for profit that focuses on bringing all forms of horticulture together. It is built as a fully functioning front end site with a CMS backend all built by hand. The file storage is handled by cloudinary while the blog styling uses froala.

You can find the github repository for it here: YoungHort Aus
